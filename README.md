## Installation

To install and run this project on your local machine, follow these steps:

1. **Clone Repository:**
   
   git clone https://github.com/punyakrit/Reachinbox-Assignment.git

   run --> cd Reachinbox-Assignment.git

   run --> npm install

   run --> npm run dev
